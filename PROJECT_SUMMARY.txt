## ğŸš¨ Common Issues & Solutions

### Issue: ModuleNotFoundError
**Solution:** Ensure virtual environment is activated and requirements installed
```bash
source venv/bin/activate
pip install -r requirements.txt
```

### Issue: Database Connection Error
**Solution:** Check PostgreSQL is running and DATABASE_URL is correct in .env
```bash
# Test connection
psql -U postgres -d verifiable_stubs
```

### Issue: Alembic Migration Error
**Solution:** Ensure database exists and migrations haven't been corrupted
```bash
python -m alembic current
python -m alembic upgrade head
```
